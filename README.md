📊 Student Report Generator – Python Project

A simple Python-based application that manages student records using CSV files. It calculates total, average, and grade, and stores the data for future use.

📌 About The Project

This project demonstrates core Python concepts such as:

File handling using CSV
User input processing
Functions and modular programming
Conditional statements
Lists and data storage

The system allows users to add student records and view stored data efficiently.

🚀 Features
Add student details
Automatic grade calculation
Data stored in CSV file
Read and display records
Simple and beginner-friendly
🛠️ Technologies Used
Python 3
CSV Module
▶️ How to Run
python student_report.py
