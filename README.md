📊 Student Report Generator – Python Project

A comprehensive Python-based application designed to manage student records using CSV file storage. This project allows users to input student details, calculate total marks, average, and grades, and store the information efficiently for future retrieval.

📌 About The Project

This project is developed as a beginner-friendly application to demonstrate important Python programming concepts such as:

File handling using CSV files
User input handling and validation
Conditional statements for grade calculation
Functions for modular programming
Lists for storing and processing marks

The system simulates a basic student management tool where data is stored persistently and can be accessed later.

🎯 Key Functionalities
Accepts student name and marks
Automatically calculates total and average
Assigns grades based on performance
Stores records in a CSV file
Reads and displays stored records
🚀 Features
Persistent data storage using CSV
Simple and interactive command-line interface
Efficient data processing
Beginner-friendly structure
Real-world application concept
🛠️ Technologies Used
Python 3
Built-in CSV module
▶️ How to Run the Program

Ensure Python is installed on your system.

Run the following command:

python student_report.py

The program will prompt you to enter student details and will store them in a CSV file.
